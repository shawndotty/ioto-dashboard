# IOTO Dashboard (IOTO 仪表盘)

IOTO Dashboard 是一个为 Obsidian 设计的插件，旨在可视化和管理您的 IOTO (Input 输入, Output 输出, Task 任务, Outcome 成果) 工作流。它提供了一个集中的仪表盘，用于跟踪您在知识管理过程中不同阶段的笔记和任务。

[English Documentation](README.md)

## 功能特点

- **工作流导航**：在输入 (Input)、输出 (Output) 和成果 (Outcome) 阶段之间轻松切换。
- **双视图模式**：
    - **笔记视图**：浏览特定工作流文件夹中的文件，显示元数据（项目、日期）。
    - **任务视图**：从专用任务文件夹聚合任务，并根据特定标题进行分类。
- **高级过滤**：
    - 按名称/内容过滤。
    - **项目过滤**：支持对库中现有项目属性的自动补全。
    - **日期过滤**：按创建或修改日期进行范围过滤。
    - **任务状态**：按已完成/未完成状态过滤任务。
- **多语言支持**：支持英语、简体中文和繁体中文。

## 配置

进入 **设置 > IOTO Dashboard** 配置您的文件夹路径：

- **输入文件夹 (Input)**：默认为 `1-Input`
- **输出文件夹 (Output)**：默认为 `2-Output`
- **成果文件夹 (Outcome)**：默认为 `4-Outcome`
- **任务文件夹 (Task)**：默认为 `3-Task` (存放您日常任务笔记的文件夹)

## 使用方法

1. 点击左侧边栏的 **Dashboard** 图标。
2. 或使用命令面板 (Ctrl/Cmd + P) 搜索 **"IOTO Dashboard: Open Dashboard"**。

## 任务聚合逻辑

插件会扫描您的 **任务文件夹** 中的文件，并查找特定标题以对任务进行分类：

- **输入 (Input) 类别**：查找 `## 输入 LEARN` 标题下的任务。
- **输出 (Output) 类别**：查找 `## 输出 THINK` 标题下的任务。
- **成果 (Outcome) 类别**：查找 `## 成果 DO` 标题下的任务。

## 开发

本插件旨在简化 Obsidian 中的 IOTO 工作流实施。

### 安装

1. 克隆此仓库。
2. 运行 `npm i` 安装依赖。
3. 运行 `npm run build` 构建插件。
